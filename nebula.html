Nebula Virtual Assistant

This project was initially crafted for the App State AppHack2024 event, a 24-hour programming competition. The original concept was to develop a virtual assistant kind of like Alexa or Google Assistant. However, instead of performing a standard Google search, it would search Stack Overflow. This modification aimed to deliver more pertinent results tailored to tech-savvy individuals and programmers.

Some features that Nebula has include but are not limited to are:

  • StackOverflow search
  • Speech to text inputs
  • Text to Speech output
  • CLI mode (no TTS or STT)
  • Weather info for the current location
  • Tells jokes
  • Pulls up YouTube videos
  • Gives dictionary definitions
  • News headlines
  • *And more*
